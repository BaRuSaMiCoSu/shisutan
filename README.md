# shisutan — シス単 単語テストメーカー

システム英単語レベルの英単語リストから、ランダムな単語テストを作成できる静的Webアプリです。
`index.html` をブラウザで開くだけで動作します(サーバー不要)。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く
2. 出題範囲(ステージ・番号)、出題形式、問題数を設定
3. 「テストを作成する」でプレビュー表示
4. 「PDFをダウンロード」でPDF保存(または「印刷」からブラウザの[PDFに保存]も利用可)

## 機能

### オンライン学習(自動採点・SRS)

- **ブラウザ内で解答**: 4択(英→日)または入力(日→英)で出題、1問ごとに自動採点
- **弱点復習リスト**: 間違えた単語を自動で蓄積し、いつでも集中復習できる
- **SRS(間隔反復)復習**: 間違えた単語は3日後→1週間後→3週間後→2か月後…の間隔で自動的に再出題。正解するたびに出題間隔が延びる
- **習得度表示**: ステージ別に「習得度 68%(410/600)」のように進捗を表示
- **学習履歴・進捗グラフ**: 何月何日に何点取ったかを記録し、直近1週間の正答率を棒グラフで可視化(すべてブラウザの localStorage に保存)

### テスト印刷

- **内蔵単語リスト**: シス単レベル準拠の約3000語(Stage 1〜4)
- **カスタムリスト取込**: 単語一覧サイト(例: ukaru-eigo.com のシス単一覧表)の表をコピーして
  貼り付けると、そのリストから出題できます(ブラウザに保存されます)
- **出題形式**: 英→日(記述)/ 日→英(記述)/ 英→日(4択)
- **解答ページ**: テストとは別ページで解答一覧を自動生成
- **PDF出力**: html2pdf.js によりHTML表示をそのままPDF化するため、日本語が文字化けしません
  (PDFライブラリはCDNから読み込むため、PDFダウンロードにはインターネット接続が必要です。
  オフライン時は「印刷」→「PDFに保存」をご利用ください)

## 技術構成

- HTML / CSS / JavaScript のみ(単一ファイル、UTF-8)
- PDF生成: [html2pdf.js](https://github.com/eKoopmans/html2pdf.js)(CDN読み込み)
